# COVID-19 Prediction System

A Machine Learning project that predicts Total COVID-19 Cases based on:

- Active Cases
- Discharged Cases
- Deaths
- Active Ratio
- Discharge Ratio
- Death Ratio
- Population

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Flask
- HTML
- CSS
- JavaScript

## How to Run

pip install -r requirements.txt

python src/train.py

python app.py
